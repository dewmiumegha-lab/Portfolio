📋 Project Overview

Run & Think is a 3D infinite-runner style game developed in Unity using C#. The player controls a character who must run, jump, and dodge obstacles — but with a twist: to progress or gain power-ups, players must answer math and logic questions correctly. Wrong answers result in obstacles or penalties.

The game was designed to make learning engaging for younger audiences by embedding educational content inside an exciting gameplay loop.

🎯 Concept & Purpose

The idea behind Run & Think was to tackle a common problem: students find traditional learning boring. By embedding quiz questions into a fast-paced runner game, the goal was to create a situation where players are motivated to think quickly and correctly — making knowledge recall feel natural and rewarding.

The game targets primary and lower secondary school students and focuses on mental arithmetic and logical reasoning skills.

🛠️ Development Process

The game was built entirely in Unity using C# scripts. Key systems developed include:

🎮 Player Controller

Handles movement, jumping, sliding, and collision detection using Unity's Rigidbody physics.

🔀 Level Generator

Procedurally generates track segments and obstacles to create an infinite play experience.

❓ Quiz System

Randomly selects and displays math questions; validates answers and triggers game events accordingly.

🏆 Score Manager

Tracks and displays player score based on distance, correct answers, and speed milestones.
