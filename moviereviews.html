📋 Project Overview

A fully functional Movie Review Web Application built using HTML, CSS, and JavaScript, backed by a Google Firebase Cloud Firestore NoSQL database. The app allows users to add, edit, delete, and sort movie reviews in real time — with no backend server or user authentication required.

🗄️ Firestore Database Structure

Data is stored in a movies collection. Each document represents one movie review with four fields:

Firestore — movies collection movies/ └── {auto-generated document ID} name : "Titanic" rating : 4 director : "James Cameron" date : "1997-12-19"

Documents are auto-ID'd by Firestore. The orderBy() query is used to sort the collection on any of the four fields before rendering.

⚙️ How It Works

The app uses Firebase's JavaScript SDK loaded via CDN as ES modules. On page load, loadMovies() queries the Firestore movies collection with orderBy() and renders each document as a list item. The Add Movie button calls addDoc() to write a new document. Edit uses updateDoc() and Delete uses deleteDoc() — all in real time with no page reload needed.
